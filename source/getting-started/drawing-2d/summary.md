# 本章小结

本章我们重点介绍了光栅化的概念，了解了如何画直线、多边形，如何填充颜色或图像，如何将各种形状组合成一幅画。尽管我们的讨论集中在二维图形中，涉及到光栅化算法、插值方法，UV 映射、深度缓存这些概念与后面要了解的三维渲染关系非常大。
